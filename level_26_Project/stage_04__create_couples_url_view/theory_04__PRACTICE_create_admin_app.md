В ТЗ есть функционал, о котором мы пока не упоминали.  
Это Админ-панель (`/admin/`).

Та прописан определённый функционал, который логично будет вывести в  
отдельное самостоятельное приложение.  
Тем более что и префикс там свой отдельный: `admin/`.

Соответственно, стандартную ссылку на админ-панель 

```python
    path('admin/', admin.site.urls),
```

мы переименуем в `administraror`:

```python
    path('administrator/', admin.site.urls),
```

### Создаём приложение `adminapp`

Назовём его безопасно (`adminapp`), чтобы не было конфликта в `admin`.

```bash
./manage.py startapp adminapp
```

```python
INSTALLED_APPS = [
    ...
    'adminapp',
]
```
